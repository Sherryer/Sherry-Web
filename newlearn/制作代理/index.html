学习地址 思路

第一步 安装 python

第二部 安装 shadowsocks包 pip install shadowsocks

第三部 vi /etc/shadowsocks.json

输入以下内容 然后 ssserver -c 

    {
        "server":"my_server_ip",
        "server_port":8888,
        "local_address": "127.0.0.1",
        "local_port":1080,
        "password":"mypassword",
        "timeout":300,
        "method":"aes-256-cfb",
        "fast_open": false
    }

    server: 服务器ip地址
    server_port: 绑定的端口，注意不要设置已经使用了的端口
    possword: 密码
    timeout: 超时时间
    method: 加密方法
    fast_open: 如果你的服务器 Linux 内核在3.7+，可以开启 fast_open 以降低延迟
    workers: 默认为1
